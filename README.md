# English-card
双语认读卡片助手 - 使用说明

## 主要功能
这是一个帮助您学习双语词汇的智能工具，通过翻转卡片的方式加深记忆，自动记录学习进度。


## 使用方法
在输入框中填写英文-中文对照内容（每行一个）
支持多种分隔格式：竖线"|"、双冒号"::"、制表符或空格
点击"生成学习卡片"开始学习
点击卡片翻转查看中文翻译
根据自身掌握情况选择"已掌握"或"未掌握"
系统会自动安排未掌握卡片进入下一轮复习

## AI辅助输入
点击"一键复制AI提示词"按钮，可将专业提示词复制到剪贴板，然后发送给AI助手（如Kimi、豆包等），AI会自动将您的学习材料转换为标准格式。


## 导入认读过关纸
点击"导入认读过关纸"按钮，可直接加载预置的Unit 1-8单元内容，无需手动输入。


## 撤销选择功能
如果您误选了"已掌握"或"未掌握"，可以点击"撤销选择"按钮回到上一个单词，重新进行选择。支持快捷键 Ctrl+Z。


## 结束学习选项
学到一半想暂停时，点击"结束学习"按钮，可选择重新学习全部内容、仅学习未掌握词汇，或返回主页。


## 学习建议
对于难记的词汇，可多次标记"未掌握"进行重复练习
善用"随机排序"功能，避免顺序记忆
可导入课本、PDF、截图等多种学习材料
⌨️
快捷键
空格键 - 翻转卡片
1 - 标记为已掌握
2 - 标记为未掌握
Esc - 翻转回正面
Ctrl+Z - 撤销上次选择
